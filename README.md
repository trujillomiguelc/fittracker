# Body Recomp Hub — Fitness Tracker

Personal fitness and nutrition tracker built with React.

## Run locally
```bash
npm install
npm run dev
```

## Deploy to GitHub Pages
```bash
npm install
npm run deploy
```

Your app will be live at: `https://YOUR_GITHUB_USERNAME.github.io/fittracker/`

## Setup steps
1. Create a GitHub repo named `fittracker`
2. Run `npm install`
3. Edit `vite.config.js` — change `base: '/fittracker/'` to match your repo name
4. Run `npm run deploy`
